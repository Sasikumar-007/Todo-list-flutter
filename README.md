# Flutter To-Do List App

A simple and user-friendly To-Do List application built using Flutter. This app helps users manage daily tasks efficiently by allowing them to add, complete, and delete tasks.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive UI
- Works on Android, iOS, Web, Windows, macOS, and Linux

## 🛠️ Technologies Used

- Flutter
- Dart
- Material Design
